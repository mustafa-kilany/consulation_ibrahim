# Strategic Consulting Website

A professional consulting website built with Nuxt.js (Vue 3) and Bootstrap 5, showcasing strategic consulting services, feasibility studies, and project analysis.

## Features

- **Modern Design**: Built with Nuxt.js and Bootstrap 5 for a responsive, professional appearance
- **Real Project Images**: Incorporates actual project images from consultation office work
- **Strategic Focus**: Highlights expertise in feasibility studies, strategic planning, and cost-benefit analysis
- **Professional Portfolio**: Showcases real projects including airport development, waterfront tourism, and construction analysis

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm run preview
```
